# Screenshots Directory

Place your application screenshots in this directory.

## Required Screenshots

1. **landing-page.png** - Landing/home page
2. **car-list.png** - Car browsing page with filters
3. **configuration.png** - Configuration page with duration and KM package selection
4. **review.png** - Review page with booking details and payment
5. **dashboard.png** - User dashboard with bookings
6. **authentication.png** - Login/signup pages
7. **confirmation.png** - Booking confirmation page

## Image Guidelines

- **Format**: PNG or JPG
- **Recommended Size**: 1920x1080 or similar
- **File Size**: Keep under 1MB per image if possible
- **Naming**: Use lowercase with hyphens (e.g., `landing-page.png`)

## Tips for Screenshots

- Use browser DevTools to capture full page screenshots
- Consider using tools like:
  - Browser extensions (Full Page Screen Capture)
  - macOS: Cmd+Shift+4 for selection, Cmd+Shift+3 for full screen
  - Windows: Snipping Tool or Win+Shift+S
- Crop to remove unnecessary browser UI
- Ensure text is readable
- Use consistent browser window size for all screenshots
